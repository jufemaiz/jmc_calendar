# Calendar

Stephen Pecoraro fleshed out this design for a calendar print sometime back. At
the time I quite liked it and worked with him in taking it from the physical
world to the digital.

Resurrected now on Ruby 2.6.1 to serve up different years along with the current
day, the previous month and the coming 10 months (to provide a rolling twelve
month view).
